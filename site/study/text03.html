3. CSS

Figma를 이용해 만든 디자인을 보며 css설정을 합니다. display: flex;와 justify-content: space-between;를 사용하여 정렬을 맞춰줍니다. n줄 효과도 넣어줍니다. 

/* fonts */
@import url('https://webfontworld.github.io/score/SCoreDream.css');

.score {
    font-family: 'SCoreDream';
    font-weight: 400;
}

/* reset */
* {
    margin: 0;
    padding: 0;
}

h2,
h3,
h4,
h5,
h6 {
    font-weight: normal;
}
