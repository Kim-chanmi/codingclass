  • 03. 숫자 메서드 : Number.MAX_SAFE_INTEGER : 안전한 최대 정수 값을 나타냄

  • 06. 숫자 메서드 : Number.isFinite() : 전달된 값이 유한 숫자인지 여부를 결정
  • 07. 숫자 메서드 : Number.isInteger() : 전달된 값이 정수인지 여부를 결정
  • 08. 숫자 메서드 : Number.isNaN() : 전달된 값이 NaN이고 해당 유형이 Number인지 확인

  • 10. 숫자 메서드 : Number.prototype.toPrecision() : 숫자를 지정된 길이로 형식화. 필요한 경우 지정된 길이를 생성하기 위해 소수점과 null이 추가됨
  • 11. 숫자 메서드 : Number.prototype.toString() : 지정된 숫자 값을 나타내는 문자열을 반환

  • 02. 날짜 메서드 : Date.now() : UTC 기준으로 1970년 1월 1일 0시 0분 0초부터 현재까지 경과된 밀리 초를 반환
  • 02. 날짜 메서드 : Date.parse() : 날짜의 문자열 표현을 구문 분석하고 문자열이 인식되지 않거나 경우에 따라 잘못된 날짜 값이 포함된 경우 1970년 1월 1일 00:00:00 UTC 또는 NaN 이후의 밀리초 수를 반환
  • 02. 날짜 메서드 : Date.UTC() : Date 생성자와 유사한 매개변수를 허용하지만 UTC로 처리. 1970년 1월 1일 00:00:00 UTC 이후의 밀리초 수를 반환
  • 02. 날짜 메서드 : Date.prototype.getFullYear() : 현지 시간에 따라 지정된 날짜의 연도를 반환
  • 02. 날짜 메서드 : Date.prototype.setFullYear() : 현지 시간에 따라 지정된 날짜의 전체 연도를 설정
  • 02. 날짜 메서드 : Date.prototype.getMonth() : 현지 시간에 따라 지정된 날짜의 월을 0부터 시작하는 값으로 반환
  • 02. 날짜 메서드 : Date.prototype.setMonth() : 현재 설정된 연도에 따라 지정된 날짜의 월을 설정
  • 02. 날짜 메서드 : Date.prototype.getDate() : 현지 시간에 따라 지정된 날짜의 날짜를 반환
  • 02. 날짜 메서드 : Date.prototype.setDate() : 현지 시간을 기준으로 지정된 Date 인스턴스의 날짜를 변경
  • 02. 날짜 메서드 : Date.prototype.getDay() : 현지 시간에 따라 지정된 날짜의 요일을 반환
  • 02. 날짜 메서드 : Date.prototype.getHours() : 현지 시간에 따라 지정된 날짜의 시간을 반환
  • 02. 날짜 메서드 : Date.prototype.setHours() : 현지 시간에 따라 지정된 날짜의 시간을 설정하고 1970년 1월 1일 00:00:00 UTC부터 업데이트된 Date 인스턴스가 나타내는 시간까지의 밀리초 수를 반환
  • 02. 날짜 메서드 : Date.prototype.getMinutes() : 현지 시간에 따라 지정된 날짜의 분을 반환
  • 02. 날짜 메서드 : Date.prototype.setMinutes() : 현지 시간에 따라 지정된 날짜의 분을 설정
  • 02. 날짜 메서드 : Date.prototype.getSeconds() : 현지 시간에 따라 지정된 날짜의 초를 반환
  • 02. 날짜 메서드 : Date.prototype.setSeconds() : 현지 시간에 따라 지정된 날짜의 초를 설정
  • 02. 날짜 메서드 : Date.prototype.getMilliseconds() : 현지 시간에 따라 지정된 날짜의 밀리초를 반환
  • 02. 날짜 메서드 : Date.prototype.setMilliseconds() : 현지 시간에 따라 지정된 날짜의 밀리초를 설정
  • 02. 날짜 메서드 : Date.prototype.getTime() : 표준시에 따라 지정된 날짜의 시간에 해당하는 숫자 값을 반환
  • 02. 날짜 메서드 : Date.prototype.setTime() : Date 객체를 1970년 1월 1일 00:00:00 UTC 이후의 시간(밀리초)으로 설정
  • 02. 날짜 메서드 : Date.prototype.getTimezoneOffset : UTC 표준 시간대에서 평가된 날짜와 현지 표준 시간대에서 평가된 동일한 날짜 간의 차이(분)를 반환
