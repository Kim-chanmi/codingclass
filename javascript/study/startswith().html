문자열 Method[startsWith(), endsWith()]

오늘은 startsWith(), endsWith()메서드에 대해 알아보겠습니다.

1. startsWith() / endsWith()

startsWith()메서드는 시작하는 문자열에서 문자열을 검색하여 불린(true, false)으로 반환합니다. endsWith()메서드는 끝나는 문자열에서 문자열을 검색하여 불린(true, false)으로 반환합니다.

"문자열".startsWith(검색 문자열);
"문자열".startsWith(검색 문자열, 위치값);
"문자열".endsWith(검색 문자열);
"문자열".endsWith(검색 문자열, 시작 위치값);

* script

const str1 = "javascript reference";
const currentStr1 = str1.startsWith('javascript'); 
const currentStr2 = str1.startsWith('j'); 
const currentStr3 = str1.startsWith('java'); 
const currentStr4 = str1.startsWith('reference'); 
const currentStr5 = str1.startsWith(); 
const currentStr6 = str1.startsWith(''); 
const currentStr7 = str1.startsWith('reference', 7); 
const currentStr8 = str1.startsWith('reference', 11); 

const currentStr9 = str1.endsWith('reference'); 
const currentStr10 = str1.endsWith('e'); 
const currentStr11 = str1.endsWith('refer'); 
const currentStr12 = str1.endsWith('javascript'); 
const currentStr13 = str1.endsWith(); 
const currentStr14 = str1.endsWith(''); 
const currentStr15 = str1.endsWith('reference', 7); 
const currentStr16 = str1.endsWith('reference', 20);
