프로그래머스 1단계 : 직사각형 별찍기

📝 문제 설명

이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다. 별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

👉 제한 조건

- n과 m은 각각 1000 이하인 자연수입니다.

👉 입출력 예

5 3

*****
*****
*****

✌️ 나의 풀이

2중 for문을 이용하여 풀이하였습니다. 주석 참고해주세요~! 
process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    for(let i=0; i<b; i++){          // b줄 만큼 반복
        let star = "";                  // 출력할 변수
        
        for(let j=0; j<a; j++){      // a개 만큼 반복
            star = star + "*";          // 출력할 변수에 "*" 입력
        }
        console.log(star)               // 출력
    }
});

👍 다른 풀이

repeat()을 이용해서 몇 줄 반복인지만 for문 돌렸네요. 똑똑이👍 
process.stdin.setEncoding('utf8');
    process.stdin.on('data', data => {
        const n = data.split(" ");
        const a = Number(n[0]), b = Number(n[1]);
        const row = '*'.repeat(a)
        for(let i =0; i < b; i++){
            console.log(row)
        }
});

👍 다른 풀이2

repeat()을 두 번 사용한 풀이방법입니다. \n은 줄바꿈 하는 방법입니다. 
process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    console.log((('*').repeat(a)+`\n`).repeat(b))
});
