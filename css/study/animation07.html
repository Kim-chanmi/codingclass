CSS

애니메이션을 위한 css입니다. 
body {
    background-color: #ffdde1;
}
.loader {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    width: 100px;
    height: 100px;
    animation: spin .6s linear infinite reverse;
}
    
.ball {
    position: absolute;
    top: 50%;
    left: 50%;

    position: absolute;
    content: '';
    background-color: #ee9ca7;
    width: 5px;

    border-radius: 100%;
    top: 0;
}
