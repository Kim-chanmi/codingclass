시멘틱 마크업

사전적 뜻은 '의미론적인'으로 해석할 수 있는데, 마크업을 할 때 의미에 부합하는 태그를 사용하라는 뜻. 시각장애인의 경우 HTML 문서의 콘텐츠 저보를 음성으로 전달해 주어야 하는데 이런 경우 전달하려는 태그의 의미가 적절해야 함

잘못된 예시

<!DOCTYPE HTML> 
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>시멘틱 마크업</title>
</head>
<body>
    반응형웹
</body>
</html>

반응형웹이 HTML 문서에서 제목으로 사용된다고 가정할 경우 예시처럼 마크업을 하면, 제목인지를 알 수가 없음


올바른 예시

<!DOCTYPE HTML> 
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>시멘틱 마크업</title>
</head>
<body>
    <h1>반응형웹</h1>
</body>
</html>

이 부분을 '반응형웹' 헤딩 일이라고 읽어주면 '제목이구나.'를 알 수 있음. 가급적 시멘틱하게 마크업을 하려고 노력하는 것이 중요함
