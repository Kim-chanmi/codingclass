HTML 기본 문서

태그(tag) : <html>, <head>, <body> 등과 같이 명령어의 형태가 '<>'로 되어있는 것
요소(element) : <body>~</body>와 같이 태그의 시작과 종료
마크업(markup) : 위의 것들로 코딩한 것

HTML 기본 문서의 <html> 요소는 <head> 요소와 body 요소로 구성되어 있으며, head 요소 안에는 meta, title, link, style, script 등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문자의 작성자 등을 설정
<title> HTML 문서의 제목을 설정
<link> 외부의 CSS 파일을 HTML 문서에 연결할 때 사용
<style> HTML 문서 내에서 스타일(CSS)을 정의함
<script> 자바스크립트를 HTML 문서 내부에서 정의하거나 외부에서 자바스크립트 파일을 불러올 때 사용

예시 1-1


    <!DOCTYPE html> 
    <html lang="ko"> 
    <head> 
        <meta charset="UTF-8"> 
        <title>Document</title> 
    </head> 
    <body> 
    </body> 
    </html>

  • DOCTYPE 선언 : 문서의 형식을 브라우저에게 알려 줌으로써 브라우저가 HTML 문서의 내용을 올바르게 표시할 수 있도록 함
  • 속성 : <html lang="ko">나 <meta charset="UTF-8">처럼 태그 내부에 부가적인 설정값을 선언한 것
  • lang 속성 : HTML 문서에서 꼭 정의해야 하는 속성. 시각장애인들이 사용하는 스크린 리더에서 언어 설정에 따라 다르게 읽어줌
  • 문자 인코딩(charset) 방식 : <meta> 태그의 charset 속성의 값 UTF-8. UTF-8의 경우 거의 대부분의 문자를 처리할 수 있어 주로 사용됨

<meta name = "viwport" />

뷰포트라고 하면 모바일 해상도에 맞게문서를 조절해 주는 속성이다.
